IT Security

Sebuah layanan untuk menjaga keamanan data operasional perusahaan baik dari internal maupun eksternal. Layanan ini mencakup analisa rules dalam penggunaan PC, Password, otorisasi akses data / system dari berbagai source yang dimungkinkan terjadi akses legal dari berbagai pihak yaitu dari sisi network, database server, web application dll nya. Kami akan memberikan hasil analisa, test security (simulasi hack system), rekomendasi serta melakukan implementasi yang dibutuhkan untuk lebih terjaganya system keamanan data di perusahaan anda.
