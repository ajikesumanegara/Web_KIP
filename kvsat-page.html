KVSAT (Kharisma VSAT)

Kenapa Harus KVSAT?

KVSat adalah solusi tepat bagi pelanggan yang membutuhkan koneksi yang stabil serta ingin membangun komunikasi dengan kantor-kantor cabangnya di berbagai pelosok daerah. Dan berikut 2 layanan kami :

1. Kharisma SPSC VSAT

Kharisma SCPC VSAT adalah solusi tepat untuk kebutuhan komunikasi data yang membutuhkan koneksi dengan dedicated bandwith, yaitu jaringan yang dikhususkan hanya untuk bisnis anda. Dimanapun kantor anda berada, teknologi Kharisma SCPC VSAT memastikan koordinasi dan komunikasi bagi bisnis anda yang stabil dan efektif.

Keunggulan Produk :

  1. Fitur Utama

    Point to Point (Private), dengan konfigurasi kedua sisi Antena VSAT yang ditetapkan di tempat pelanggan.

  2. Fitur Tambahan

    Video Conference & Internet.

  3. Spesifikasi
    • Data rate at 64 Kbps - 2048 Kbps or more
    • Clear channel configuration through where a communication protocol such as: Metro Ethernet, can run
    • Full duplex configuration with modulation: QPSK/8PSK/8 APSK/16 QAM

Paket Layanan :

  1. SCPC VSAT Private
  2. Sea Star Private
  3. SCPC VSAT Teleport
  4. Sea Star Teleport

2. Kharisma IP VSAT

Kharisma IP VSAT adalah solusi tepat untuk kebutuhan komunikasi data antar kantor cabang dengan memanfaatkan satelit. Solusi Lintasarta IP VSAT membantu perusahaan menjangkau cabang bisnisnya bahkan di lokasi terpencil sekalipun.

Keunggulan Produk :

  1. Aplikasi Transaksional dan Interaktif
    • Online communication among branches
    • Hotel/flight reservation
    • ATM (Automated Teller Machine)
    • Low data traffic
    • Voice over IP (VoIP)
  2. Aplikasi Remote Terminal / Telnet / Terminal Emulation dengan Sentralisasi Database
    • Data entry
    • Inventory control
    • Payment point
  3. Web Surfing
    • E-mail
    • Instant messaging
    • File Transfer Protocol (FTP)
  4. Aplikasi Remote dengan Download Besar
    • Internet
    • Distance-Learning

Paket Layanan :

  1. IP VSAT Low Traffic Priority
  2. IP VSAT Low Traffic Business
  3. IP VSAT Broadband Downstream ½
  4. IP VSAT Broadband Downstream ¼
  5. IP VSAT Broadband Upstream
  6. IP VSAT Nomadic
